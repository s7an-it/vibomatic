# Design System: Todo API

> Minimal design tokens for the todo-api example.

## Colors

| Token | Value | Usage |
|-------|-------|-------|
| `color.primary` | blue-600 | Primary actions, links |
| `color.neutral` | gray-500 | Secondary actions, borders |
| `color.status.pending` | yellow-500 | Pending todo badge |
| `color.status.done` | green-600 | Completed todo badge |
| `color.status.overdue` | red-600 | Overdue todo badge |
| `color.bg` | white | Page background |
| `color.text` | gray-900 | Primary text |

## Typography

| Token | Size | Weight | Usage |
|-------|------|--------|-------|
| `type.heading.sm` | 18px | 600 | Modal headings, section titles |
| `type.body.md` | 14px | 400 | Card text, form labels |
| `type.caption` | 12px | 400 | Timestamps, secondary info |

## Spacing

| Token | Value | Usage |
|-------|-------|-------|
| `spacing.xs` | 4px | Inline padding |
| `spacing.sm` | 8px | Badge padding, icon gaps |
| `spacing.md` | 16px | Card gaps, section margins |
| `spacing.lg` | 24px | Page padding |
